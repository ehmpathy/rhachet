# 3.2.distill.repros.experience._.v1.i1

## experience reproductions

| experience | entry point | user actions | expected outcome | test type |
|------------|-------------|--------------|------------------|-----------|
| fill all keys for default owner | CLI | run `rhx keyrack fill --env test` | all keys set, unlocked, verified | integration |
| fill all keys for multiple owners | CLI | run `rhx keyrack fill --env test --owner default --owner ehmpath --prikey ~/.ssh/ehmpath` | keys set for both owners | integration |
| partial fill (some keys already set) | CLI | run fill when 2/3 keys already set | skip already-set, prompt for 1 absent | integration |
| refresh all keys | CLI | run `rhx keyrack fill --env test --refresh` | re-prompt and overwrite all | integration |
| fill specific key only | CLI | run `rhx keyrack fill --env test --key CLOUDFLARE_API_TOKEN` | only that key filled | integration |
| fail-fast: no prikey for owner | CLI | run fill with owner whose prikey is unavailable | error: "no available prikey for owner=X" | unit |
| fail-fast: key not found | CLI | run fill with `--key NONEXISTENT` | error: "key X not found in manifest for env=Y" | unit |

## reproduction feasibility

### test utilities available

- `genMockKeyrackRepoManifest` — create test manifest
- `genMockKeyrackHostManifest` — create test host manifest
- `jest.mock` for daoKeyrackHostManifest, daemon sdk
- `promptHiddenInput` can be mocked to return test values

### setup required

- mock repo manifest with test keys
- mock host manifest (empty for fresh fill, partial for skip tests)
- mock prikey discovery to return test identity
- mock prompt to return test secret

### test sketch

```ts
describe('fillKeyrackKeys', () => {
  given('[case1] repo manifest with 2 keys, owner=default', () => {
    const repoManifest = genMockKeyrackRepoManifest({
      keys: {
        'ehmpathy.test.CLOUDFLARE_API_TOKEN': { env: 'test', ... },
        'ehmpathy.test.CLOUDFLARE_ACCOUNT_ID': { env: 'test', ... },
      },
    });
    const hostManifest = genMockKeyrackHostManifest({ hosts: {} });

    when('[t0] before fill', () => {
      then('hostManifest is empty', () => {
        expect(Object.keys(hostManifest.hosts)).toHaveLength(0);
      });
    });

    when('[t1] fillKeyrackKeys is called', () => {
      const result = useThen('fill succeeds', async () =>
        fillKeyrackKeys({ env: 'test', owners: ['default'] }, context),
      );

      then('all keys are set', () => {
        expect(result.results).toHaveLength(2);
        expect(result.results.every(r => r.status === 'set')).toBe(true);
      });
    });
  });
});
```
