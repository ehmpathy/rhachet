# blocker.1: domain.objects depends on domain.operations (upward dependency)

**rule**: .agent/repo=bhuild/role=behaver/briefs/practices/behavior.execution/architect/rule.forbid.scope-leaks.md

**locations**:
- src/domain.objects/keyrack/KeyrackHostVaultAdapter.ts:1

KeyrackHostVaultAdapter.ts imports ContextKeyrack from @src/domain.operations/keyrack/genContextKeyrack. This violates the directional dependency rule: domain.objects (lower layer) must not import from domain.operations (higher layer).
