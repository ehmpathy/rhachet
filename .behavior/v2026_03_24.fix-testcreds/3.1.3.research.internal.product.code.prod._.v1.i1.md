### location

`src/contract/cli/invokeKeyrack.ts:925-1023`

### relevance

### citations

## pattern.4 = getAllAvailableIdentities

### location

`src/access/daos/daoKeyrackHostManifest/index.ts:57-93`

### relevance

this is the function that needs extension to include `~/.ssh/$owner` discovery path.

### citations

[8] current discovery paths (lines 57-65):
```ts
const getAllAvailableIdentities = (): string[] => {
  const identities: string[] = [];
  const home = process.env.HOME ?? homedir();
  const standardPaths = [
    join(home, '.ssh', 'id_ed25519'),
    join(home, '.ssh', 'id_rsa'),
    join(home, '.ssh', 'id_ecdsa'),
  ];
```

[9] ssh-agent checked first (lines 67-79):
```ts
// check ssh-agent first
const agentKeys = listSshAgentKeys();
for (const agentKey of agentKeys) {
  try {
    // agent keys have path as comment (e.g., "/home/user/.ssh/id_ed25519")
    const keyPath = agentKey.comment;
    if (keyPath && existsSync(keyPath)) {
      const identity = sshPrikeyToAgeIdentity({ keyPath });
      if (!identities.includes(identity)) identities.push(identity);
    }
  } catch {
    // skip keys that fail to convert
  }
}
```

[10] standard paths checked second (lines 81-92):
```ts
// check standard ssh paths
for (const stdPath of standardPaths) {
  if (!existsSync(stdPath)) continue;
  try {
    const identity = sshPrikeyToAgeIdentity({ keyPath: stdPath });
    if (!identities.includes(identity)) identities.push(identity);
  } catch {
    // skip keys that fail to convert
  }
}
```

### mark

**[EXTEND]** — add `~/.ssh/$owner` to discovery paths when owner is known

### implementation note

the `getAllAvailableIdentities` function currently has no access to `owner`. to implement auto-discovery from `~/.ssh/$owner`:

option A: pass `owner` to `getAllAvailableIdentities` and add it to discovery paths
option B: check `~/.ssh/$owner` at the call site in `get()` before fallback to discovery

option A is cleaner — modify `getAllAvailableIdentities` to accept optional `owner` parameter:
```ts
const getAllAvailableIdentities = (owner?: string | null): string[] => {
  const identities: string[] = [];
  const home = process.env.HOME ?? homedir();

  // owner-specific path takes priority (e.g., ~/.ssh/ehmpath)
  if (owner) {
    const ownerPath = join(home, '.ssh', owner);
    if (existsSync(ownerPath)) {
      // ... try to convert and add
    }
  }

  // then standard paths...
}
```
