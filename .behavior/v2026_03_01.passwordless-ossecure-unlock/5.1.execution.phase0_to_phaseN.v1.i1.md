## phase 1: core logic changes

- [x] **1.1** update vaultAdapterOsSecure.ts unlock logic
  - check `getActiveIdentity() !== null` first -> return early
  - check `input.identity !== null` -> set sessionIdentity, return
  - removed all passphrase logic (input check, env var, prompt)
  - throws UnexpectedCodePathError if no identity

- [x] **1.2** update unlockKeyrackKeys.ts
  - changed `passphrase: input.passphrase` to `identity: null`
  - session identity flows from manifest decryption via setOsSecureSessionIdentity()

- [x] **1.3** update setKeyrackKey.ts
  - removed env=all expansion logic
  - always uses single slug: `${input.org}.${input.env}.${input.key}`

- [x] **1.4** update getAllKeyrackSlugsForEnv.ts
  - changed filter from `spec.env === input.env`
  - to `spec.env === input.env || spec.env === 'all'`

**verification**: `npm run test:types && npm run test:unit` passes
