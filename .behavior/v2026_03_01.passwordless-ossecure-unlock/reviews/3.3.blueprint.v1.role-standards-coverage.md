## coverage check

**required**: early exits and helpful errors for invalid state

**present in blueprint**:
- vaultAdapterOsSecure.ts throws `UnexpectedCodePathError` when no identity (line 177-180)
- error includes actionable hint: "use --prikey to specify ssh key or run keyrack init"
- early returns for valid identity cases (lines 150, 153-156)

### rule.forbid.undefined-attributes ✅

**required**: use `string | null` instead of `string?` for internal contracts

**present in blueprint**:

- `identity: string | null` in vaultAdapterOsSecure.unlock signature (line 142)
- callers must explicitly pass `null` (line 192)

### rule.require.input-context-pattern ✅

### rule.require.idempotent-procedures ✅

**required**: mutations should be idempotent

**present in blueprint**:
