wish =

we could execute `npx rhachet run -s $skill` and rhachet would
1. look across all of the roles from `rhachet.use.ts`
2. see if theres exactly one skill that's named after that
   3. from Role.skills.dirs/*, where the name of the file in that dir is $skill.sh or just $skill; i.e., there's an executable file with that skill name

and if so, invokes it; i.e., becomes a dispatcher which knows where to look for those skill executables

should support

- `npx rhachet run --skill $skill`
- `npx rhachet run --repo $repo --skill $skill`
- `npx rhachet run --role $role --skill $skill`
- `npx rhachet run --repo $repo --role $role --skill $skill`

e.g.,

`npx rhachet run --skill init.behavior`
`npx rhachet run --repo bhuild --skill init.behavior`
`npx rhachet run --role behaver --skill init.behavior`
