# q7: per-brain isolation — yield

## answer

**process-scoped** — each terminal/process has independent auth

## rationale

apiKeyHelper is configured in `settings.json` and called by each Claude CLI process independently:

```json
{
  "apiKeyHelper": "rhx auth-pool get-best --creds $uri"
}
```

each process:
1. reads its own `settings.json`
2. calls `apiKeyHelper` executable on its own schedule (every 5 min or on 401)
3. receives token from stdout
4. uses that token for subsequent requests

no shared state between processes — each is independent.

## implications

- rotation in one terminal does NOT affect others
- auth-pool state file is shared, but reads are atomic
- each process selects "best" token at call time
- natural load distribution across subscriptions
