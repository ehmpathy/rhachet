## phase 3: manifest init update

**prereqs:** phase 1, phase 2

**read before:**
- `.behavior/v2026_03_09.rhachet-agent-keyrack-init/3.1.research.patterns._.code.prod.v1.i1.md`
- `.behavior/v2026_03_09.rhachet-agent-keyrack-init/2.1.criteria.blackbox.md` (usecase.1, usecase.4, usecase.11)

### tasks

- [x] create `src/domain.operations/keyrack/initKeyrackRepoManifest.ts`
- [ ] update `src/domain.operations/keyrack/initKeyrackRepoManifest.ts`
  - [ ] accept required `roles: string[]` input (which roles to extend)
  - [ ] accept optional `org` input (override)
  - [ ] validate specified roles have keyracks (fail-fast if absent)
  - [ ] if org not provided → call `getOrgFromPackageJson()`
  - [ ] if org still null → throw `BadRequestError('provide --org')`
  - [ ] call `discoverRoleKeyracks({ roles })` to get extends paths for specified roles
  - [ ] if manifest extant → merge extends (findsert new, preserve extant, sort)
  - [ ] if manifest absent → create with org + extends + env sections
  - [ ] return `{ effect, org, extends }`

- [x] update `src/domain.operations/keyrack/initKeyrackRepoManifest.integration.test.ts`
- [ ] update tests for roles input and validation
  - [ ] test fresh repo with specified roles → creates with extends for those roles
  - [ ] test extant manifest → appends new extends for specified roles
  - [ ] test specified role lacks keyrack → throws BadRequestError
  - [ ] test extends sorted alphabetically
  - [ ] test only specified roles in extends (unspecified excluded)

### acceptance criteria

```
given('repo with package.json and no .agent/keyrack.yml')
  when('initKeyrackRepoManifest is called with { roles: ["tester"] }')
    then('creates .agent/keyrack.yml')
    then('org is auto-detected')
    then('extends array contains only tester keyrack')
    then('env.prod, env.prep, env.test sections are present')

given('repo with .agent/keyrack.yml that has extends entries')
  when('initKeyrackRepoManifest is called with { roles: ["tester", "newrole"] }')
    then('newrole keyrack is appended')
    then('prior extends entries are preserved')

given('repo with role that lacks keyrack')
  when('initKeyrackRepoManifest is called with { roles: ["norack"] }')
    then('throws BadRequestError')
    then('error indicates which role lacks keyrack')
```

### verification

```sh
npm run test:integration -- initKeyrackRepoManifest
```
