## summary

this research identifies extant test patterns relevant to `init --keys` feature implementation.

## pattern.1: acceptance test structure

### [REUSE] blackbox/cli/keyrack.init.acceptance.test.ts

**source:** `blackbox/cli/keyrack.init.acceptance.test.ts` [^1]

acceptance tests for keyrack init follow bdd structure with `given/when/then`:

```ts
describe('keyrack init', () => {
  given('[case1] repo with keyrack.yml', () => {
    const scene = useBeforeAll(async () => {
      const repo = await genTestTempRepo({ fixture: 'with-role-keyrack' });
      return { repo };
    });

    when('[t0] keyrack init is invoked', () => {
      const result = useThen('it succeeds', async () =>
        invokeRhachetCliBinary({
          args: ['keyrack', 'init', '--for', 'owner', ...],
          cwd: scene.repo.path,
        }),
      );

      then('exit code is 0', () => {
        expect(result.status).toEqual(0);
      });
    });
  });
});
```

**applicability:** same bdd structure for `init --keys` acceptance tests.

## pattern.2: test fixture generation

### [REUSE] genTestTempRepo

**source:** `blackbox/.test/infra/genTestTempRepo.ts` [^2]

creates isolated test repos in `os.tmpdir()`:

```ts
export const genTestTempRepo = async (input: {
  fixture: TestRepoFixture;
}): Promise<{ path: string; cleanup: () => Promise<void> }> => {
  const tempDir = await fs.mkdtemp(join(os.tmpdir(), 'rhachet-test-'));
  const fixturePath = join(__dirname, '../assets', input.fixture);
  await fs.cp(fixturePath, tempDir, { recursive: true });
  // ... git init, setup
  return { path: tempDir, cleanup };
};
```

## pattern.3: cli binary invocation

### [REUSE] invokeRhachetCliBinary

**source:** `blackbox/.test/infra/invokeRhachetCliBinary.ts` [^4]

invokes compiled cli binary as subprocess:

```ts
export const invokeRhachetCliBinary = (input: {
  args: string[];
  cwd: string;
  env?: Record<string, string>;
}): { status: number; stdout: string; stderr: string } => {
  const result = spawnSync(binaryPath, input.args, {
    cwd: input.cwd,
    env: { ...process.env, ...input.env },
    encoding: 'utf-8',
  });
  return {
    status: result.status ?? 1,
    stdout: result.stdout,
    stderr: result.stderr,
  };
};
```

**applicability:** invoke `init --keys` via this helper for acceptance tests.

### [REUSE] asSnapshotSafe

**source:** `blackbox/.test/infra/invokeRhachetCliBinary.ts` [^5]

strips machine-specific content for snapshot tests:

```ts
export const asSnapshotSafe = (input: { stdout: string }): string => {
  return input.stdout
    .replace(/\/tmp\/[^\s]+/g, '/tmp/__TEMP__')
    .replace(/\/home\/[^\s]+/g, '/home/__USER__');
};
```

**applicability:** use for output snapshot tests of `init --keys`.

## pattern.5: integration test structure

### [REUSE] initKeyrackRepoManifest.integration.test.ts

**source:** `src/domain.operations/keyrack/initKeyrackRepoManifest.integration.test.ts` [^7]

integration tests for keyrack manifest init:

```ts
describe('initKeyrackRepoManifest', () => {
  given('[case1] repo without keyrack.yml', () => {
    const scene = useBeforeAll(async () => {
      const tempDir = await fs.mkdtemp(join(os.tmpdir(), 'test-'));
      await fs.mkdir(join(tempDir, '.agent'), { recursive: true });
      return { tempDir };
    });

    when('[t0] initKeyrackRepoManifest is called', () => {
      then('creates keyrack.yml', async () => {
        const result = await initKeyrackRepoManifest(
          { org: 'test-org' },
          { gitroot: scene.tempDir },
        );
        expect(result.effect).toEqual('created');
      });
    });
  });
});
```

**applicability:** extend for test of org auto-detection and extends population.

## pattern.6: idempotency test

### [REUSE] findsert idempotency pattern

**source:** `blackbox/cli/keyrack.init.acceptance.test.ts` [^8]

tests idempotent behavior:

```ts
given('[case2] repo with extant keyrack.yml', () => {
  when('[t0] keyrack init is invoked again', () => {
    then('returns found effect', async () => {
      const result = await invokeRhachetCliBinary({ ... });
      expect(result.stdout).toContain('found');
    });

    then('keyrack.yml is unchanged', async () => {
      const contentBefore = await fs.readFile(manifestPath, 'utf-8');
      await invokeRhachetCliBinary({ ... });
      const contentAfter = await fs.readFile(manifestPath, 'utf-8');
      expect(contentAfter).toEqual(contentBefore);
    });
  });
});
```

**applicability:** same pattern for `init --keys` idempotency tests.
