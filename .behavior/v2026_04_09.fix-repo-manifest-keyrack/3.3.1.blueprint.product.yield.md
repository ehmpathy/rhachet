# blueprint: `rhachet repo compile`

## summary

create `rhachet repo compile --from src --into dist` command that:
1. discovers roles via same logic as `repo introspect` (getRoleRegistry export)
2. for each role, applies suffix globs to registered dirs (briefs.dirs, skills.dirs, inits.dirs)
3. copies matched files to dist/ with preserved relative structure
4. applies default exclusions and honors `--include`/`--exclude` overrides

### getAllArtifactsForRole

```ts
const DEFAULT_ARTIFACT_EXCLUSIONS = ['.test/**', '__test_*/**', '.route/**', '.scratch/**', '.behavior/**', '*.test.*', '.*'];

const DEFAULT_ARTIFACT_INCLUSIONS = {
  briefs: ['**/*.md', '**/*.min'],
  skills: ['**/*.sh', '**/*.jsonc', '**/template/**', '**/templates/**'],
  inits: ['**/*.sh', '**/*.jsonc'],
};

/**
 * .what = collect all artifact paths for a single role
 * .why = centralize artifact discovery logic
 */
export const getAllArtifactsForRole = async (
  input: {
    role: Role;
    fromDir: string;
    include?: string[];
    exclude?: string[];
  },
): Promise<string[]> => {
  const artifacts: string[] = [];

  // briefs from registered dirs
  for (const dir of extractDirUris(input.role.briefs?.dirs)) {
    const fullPath = path.join(input.fromDir, dir);
    if (!fs.existsSync(fullPath)) {
      throw new BadRequestError('briefs dir not found', { role: input.role.name, dir });
    }
    const files = await getAllFilesByGlobs({
      cwd: fullPath,
      defaultInclude: DEFAULT_ARTIFACT_INCLUSIONS.briefs,
      defaultExclude: DEFAULT_ARTIFACT_EXCLUSIONS,
      userInclude: input.include ?? [],
      userExclude: input.exclude ?? [],
    });
    artifacts.push(...files);
  }

  // skills from registered dirs
  for (const dir of extractDirUris(input.role.skills?.dirs)) {
    const fullPath = path.join(input.fromDir, dir);
    if (!fs.existsSync(fullPath)) {
      throw new BadRequestError('skills dir not found', { role: input.role.name, dir });
    }
    const files = await getAllFilesByGlobs({
      cwd: fullPath,
      defaultInclude: DEFAULT_ARTIFACT_INCLUSIONS.skills,
      defaultExclude: DEFAULT_ARTIFACT_EXCLUSIONS,
      userInclude: input.include ?? [],
      userExclude: input.exclude ?? [],
    });
    artifacts.push(...files);
  }

  // inits from registered dirs
  for (const dir of extractDirUris(input.role.inits?.dirs)) {
    const fullPath = path.join(input.fromDir, dir);
    if (!fs.existsSync(fullPath)) {
      throw new BadRequestError('inits dir not found', { role: input.role.name, dir });
    }
    const files = await getAllFilesByGlobs({
      cwd: fullPath,
      defaultInclude: DEFAULT_ARTIFACT_INCLUSIONS.inits,
      defaultExclude: DEFAULT_ARTIFACT_EXCLUSIONS,
      userInclude: input.include ?? [],
      userExclude: input.exclude ?? [],
    });
    artifacts.push(...files);
  }

  // role-level files (readme, boot, keyrack)
  const roleDir = path.dirname(input.role.readme);
  if (fs.existsSync(path.join(input.fromDir, roleDir, 'readme.md'))) {
    artifacts.push(path.join(roleDir, 'readme.md'));
  }
  if (fs.existsSync(path.join(input.fromDir, roleDir, 'boot.yml'))) {
    artifacts.push(path.join(roleDir, 'boot.yml'));
  }
  if (fs.existsSync(path.join(input.fromDir, roleDir, 'keyrack.yml'))) {
    artifacts.push(path.join(roleDir, 'keyrack.yml'));
  }

  return artifacts;
};
```

### getAllFilesByGlobs

```ts
import pm from 'picomatch';

/**
 * .what = get files by glob patterns with rsync-style precedence
 * .why = user includes can rescue files from default exclusions
 *
 * precedence (highest to lowest):
 *   1. user exclude → always excludes
 *   2. user include → rescues from default exclusions
 *   3. default exclude → applies unless user include matches
 *   4. default include → baseline
 */
export const getAllFilesByGlobs = async (input: {
  cwd: string;
  defaultInclude: string[];
  defaultExclude: string[];
  userInclude: string[];
  userExclude: string[];
}): Promise<string[]> => {
  // gather all candidate files (no exclusions yet)
  const allIncludes = [...input.defaultInclude, ...input.userInclude];
  const candidates = fg.sync(allIncludes, {
    cwd: input.cwd,
    absolute: true,
  });

  // build matchers
  const matchUserExclude = pm(input.userExclude, { dot: true });
  const matchUserInclude = pm(input.userInclude, { dot: true });
  const matchDefaultExclude = pm(input.defaultExclude, { dot: true });

  // apply rsync-style precedence filter
  return candidates.filter((file) => {
    const rel = path.relative(input.cwd, file);

    // 1. user exclude → always excludes
    if (input.userExclude.length && matchUserExclude(rel)) return false;

    // 2. user include → rescues from default exclusions
    if (input.userInclude.length && matchUserInclude(rel)) return true;

    // 3. default exclude → applies
    if (input.defaultExclude.length && matchDefaultExclude(rel)) return false;

    // 4. included by default
    return true;
  });
};
```

## critical: precedence semantics (rsync-style)

precedence order (highest to lowest):
1. **user `--exclude`** → always excludes, no override possible
2. **user `--include`** → overrides default exclusions, rescues files
3. **default exclusions** → apply unless user include matches
4. **default includes** → baseline patterns

this enables:
- `--include '*.test.sh'` rescues test files from default `*.test.*` exclusion
- `--exclude 'vendor/**'` always excludes, even if user includes `**/*.sh`

implementation approach:
1. gather all files via (default includes + user includes)
2. for each file, apply precedence filter:
   - if matches user exclude → exclude
   - if matches user include → include (skip default exclusion check)
   - if matches default exclusion → exclude
   - otherwise → include

## CLI interface

```
npx rhachet repo compile --from <dir> --into <dir> [--include <glob>]... [--exclude <glob>]...
```

| flag | required | description |
|------|----------|-------------|
| --from | yes | source directory (e.g., `src`) |
| --into | yes | destination directory (e.g., `dist`) |
| --include | no | additional glob patterns to include (repeatable) |
| --exclude | no | additional glob patterns to exclude (repeatable) |

## error handling

errors throw BadRequestError with context (matches extant rhachet pattern):

| error | message | context |
|-------|---------|---------|
| absent --from | `--from is required` | `{}` |
| absent --into | `--into is required` | `{}` |
| --into outside repo | `--into must be within repo` | `{ provided, repo }` |
| no package.json | `no package.json found` | `{ expected }` |
| no roles | `no roles discovered` | `{ hint }` |
| role dir absent | `briefs dir not found` | `{ role, dir }` |
