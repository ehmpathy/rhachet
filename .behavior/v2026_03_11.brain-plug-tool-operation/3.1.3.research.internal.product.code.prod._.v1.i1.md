## pattern 1: BrainPlugToolDefinition [EXTEND]

**file:** `src/domain.objects/BrainPlugToolDefinition.ts`

**current shape:**
```ts
export interface BrainPlugToolDefinition {
  slug: string;
  description: string;
  schema: {
    input: z.Schema<unknown>;
  };
}
```

**extension needed:**
- add `TInput` generic for typed input schema
- add `TOutput` generic for typed execute return
- add `TBrainGrain extends BrainGrain` type parameter
- add optional `execute` operation (present only when `TBrainGrain = 'repl'`)

**proposed shape:**
```ts
export interface BrainPlugToolDefinition<
  TInput = unknown,
  TOutput = unknown,
  TBrainGrain extends BrainGrain = BrainGrain,
> {
  slug: string;
  description: string;
  schema: {
    input: z.Schema<TInput>;
  };
  execute?: TBrainGrain extends 'repl'
    ? (input: { invocation: BrainPlugToolInvocation<TInput> }, context: unknown) => Promise<TOutput>
    : never;
}
```

## pattern 4: BrainOutput [EXTEND]

**file:** `src/domain.objects/BrainOutput.ts`

**current conditional type:**
```ts
export type AsBrainOutputCallsFor<TPlugs extends BrainPlugs> =
  TPlugs extends { tools: BrainPlugToolDefinition[] }
    ? { tools: BrainPlugToolInvocation[] } | null
    : null;
```

**extension needed:**
- add `TGrain extends BrainGrain` parameter
- repl always returns `null` for calls (executes tools internally)
- atom returns `{ tools } | null` when tools plugged

**proposed shape:**
```ts
export type AsBrainOutputCallsFor<
  TPlugs extends BrainPlugs,
  TGrain extends BrainGrain,
> = TGrain extends 'repl'
  ? null
  : TPlugs extends { tools: BrainPlugToolDefinition[] }
    ? { tools: BrainPlugToolInvocation[] } | null
    : null;
```

## new domain object: BrainPlugToolExecution [NEW]

**file:** `src/domain.objects/BrainPlugToolExecution.ts` (new file)

**proposed shape:**
```ts
import { DomainLiteral } from 'domain-objects';
import { IsoDuration } from 'iso-time';

export interface BrainPlugToolExecution<TInput = unknown, TOutput = unknown> {
  /** correlation id from invocation.exid */
  exid: string;

  /** tool identifier for validation */
  slug: string;

  /** what the tool was called with */
  input: TInput;

  /** what the tool returned */
  output: TOutput;

  /** outcome signal for brain interpretation */
  signal: 'success' | 'error:constraint' | 'error:malfunction';

  /** execution observability */
  metrics: {
    cost: {
      time: IsoDuration;
    };
  };
}

export class BrainPlugToolExecution<TInput = unknown, TOutput = unknown>
  extends DomainLiteral<BrainPlugToolExecution<TInput, TOutput>>
  implements BrainPlugToolExecution<TInput, TOutput> {}
```
