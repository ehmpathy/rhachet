# blueprint: product

---

## summary

implement `BrainPlugToolExecution<TInput, TOutput>` domain object and related type extensions to enable typed tool result continuation for brain atoms and repls.

key deliverables:
1. `BrainPlugToolExecution` domain object with exid, slug, input, output, signal, metrics
2. `BrainPlugToolDefinition` extension with generics and required execute for repl
3. `AsBrainOutputCallsFor` conditional type update to consider TGrain
4. prompt union type extension: `string | BrainPlugToolExecution[]`
5. test factories and coverage for all usecases
