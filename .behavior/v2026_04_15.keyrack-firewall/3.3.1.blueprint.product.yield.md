# 3.3.1.blueprint.product.yield — keyrack firewall

blueprint for the keyrack firewall CLI command.

---

## summary

build a CLI command that translates and validates secrets for CI environments:

```
npx rhachet keyrack firewall --env test --from 'json(env://SECRETS)' --into github.actions
npx rhachet keyrack firewall --env test --owner ehmpath --from 'json(env://SECRETS)' --into github.actions
```

**flags**:
| flag | required | default | description |
|------|----------|---------|-------------|
| `--env` | yes | — | which env to grant |
| `--from` | yes | — | input source slug |
| `--into` | yes | — | output format |
| `--owner` | no | `default` | keyrack owner |

the command:

1. reads secrets from input source (`--from`)
2. injects them into process.env
3. calls extant `keyrack get --for repo` (translates and validates)
4. outputs granted keys via firewall output format (`--into`)
5. exits 2 if any blocked

**why CLI over GitHub Action**: testability. a CLI command can be tested via normal integration tests with mocked env vars and temp files. a GitHub Action requires either `act` (flaky) or real workflow runs (slow). by pushing all logic into the CLI, we get fast, reliable tests.

**extensible design**:

| `--from` | input source |
|----------|--------------|
| `json(env://VAR)` | read JSON from env var |
| `json(stdin://*)` | read JSON from stdin |

| `--into` | firewall output format |
|----------|------------------------|
| `github.actions` | `::add-mask::` + `$GITHUB_ENV` |
| `json` | structured JSON (for tests, programmatic use) |

**key insight**: `vaultAdapterOsEnvvar.get()` should follow the same pattern as other vaults — accept `input.mech` and call `mechAdapter.deliverForGet()`. when `input.mech` is not supplied, detect mech from JSON blob in the value.

## the fix: vaults return KeyrackKeyGrant

**key insight**: vaults have all the info to produce a full `KeyrackKeyGrant`:
- `slug` — passed as input
- `key.secret` — the translated secret
- `key.grade` — computed from secret pattern (after translation)
- `source.vault` — itself
- `source.mech` — the mech used
- `env`/`org` — derivable from slug (`org.env.keyName`)
- `expiresAt` — from mech.deliverForGet

get: async (input: { slug: string; mech?: KeyrackGrantMechanism }): Promise<KeyrackKeyGrant | null> => {
  const keyName = asKeyrackKeyName({ slug: input.slug });
  const source = process.env[keyName] ?? null;
  if (source === null) return null;

  // detect mech from value (JSON blob or plain string)

      slug: input.slug,

    slug: input.slug,
    key: { secret, grade },
    source: { vault: 'os.envvar', mech },
    env,
    org,
    expiresAt,
  });

**new transformer**: `src/domain.operations/keyrack/inferKeyrackMechForGet.ts`
```typescript
import type { KeyrackGrantMechanism } from '@src/domain.objects/keyrack';

/**
 * .what = detect mech field from value (JSON blob or plain string)
 * .why = enables os.envvar vault to translate github app blobs
 */
export const inferKeyrackMechForGet = (input: {
  value: string;
}): KeyrackGrantMechanism => {
  try {
    const parsed = JSON.parse(input.value);
    if (parsed.mech && typeof parsed.mech === 'string') {
      return parsed.mech as KeyrackGrantMechanism;
    }
  } catch {
    // not JSON, passthrough
  }
  return 'PERMANENT_VIA_REPLICA';
};
```

**new transformer**: `src/domain.operations/keyrack/asKeyrackFirewallSource.ts`
```typescript
import { ConstraintError } from 'helpful-errors';

type KeyrackFirewallSourceType = 'env' | 'stdin';

interface KeyrackFirewallSource {
  type: KeyrackFirewallSourceType;
  format: 'json';
  envVar?: string; // for type: 'env'
}

/**
 * .what = parse --from slug into structured source descriptor
 * .why = enables extensible input sources for keyrack firewall CLI
 *
 * supported formats:
 *   json(env://VAR)   → read JSON from env var VAR
 *   json(stdin://*)    → read JSON from stdin
 */
export const asKeyrackFirewallSource = (input: {
  slug: string;
}): KeyrackFirewallSource => {
  // pattern: format(protocol://path)
  const match = input.slug.match(/^(\w+)\((\w+):\/\/(.*)?\)$/);
  if (!match) {
    throw new ConstraintError('invalid --from slug format', {
      slug: input.slug,
      hint: "expected format: json(env://VAR) or json(stdin://*)",
      examples: [
        "json(env://SECRETS)",
        "json(stdin://*)",
      ],
    });
  }

  const [, format, protocol, path] = match;

  // validate format
  if (format !== 'json') {
    throw new ConstraintError('unsupported --from format', {
      format,
      supported: ['json'],
      hint: "only json format is supported",
    });
  }

  // validate protocol
  if (protocol === 'env') {
    if (!path) {
      throw new ConstraintError('env:// requires variable name', {
        slug: input.slug,
        hint: "use json(env://VAR_NAME)",
      });
    }
    return { type: 'env', format: 'json', envVar: path };
  }

  if (protocol === 'stdin') {
    return { type: 'stdin', format: 'json' };
  }

  throw new ConstraintError('unsupported --from protocol', {
    protocol,
    supported: ['env', 'stdin'],
    hint: "use json(env://VAR) or json(stdin://*)",
  });
};
```

import { asKeyrackKeyOrg } from './asKeyrackKeyOrg';
import { asKeyrackKeyName } from './asKeyrackKeyName';

/**
 * .what = extract org, env, and keyName from slug
 * .why = vaults need these parts to construct KeyrackKeyGrant
 */
export const asKeyrackSlugParts = (input: {
  slug: string;
}): { org: string; env: string; keyName: string } => {
  // slug format: $org.$env.$keyName
  const parts = input.slug.split('.');
  const org = asKeyrackKeyOrg({ slug: input.slug });
  const env = parts[1] ?? '';
  const keyName = asKeyrackKeyName({ slug: input.slug });
  return { org, env, keyName };
};

## workflow usage

no action.yml needed — use the CLI directly in workflow steps:

```yaml
# .github/workflows/test.yml
jobs:
  test:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-node@v4
        with:
          node-version: '20'
      - run: npm ci

      # keyrack firewall: translate and validate secrets
      - run: npx rhachet keyrack firewall --env test --owner ehmpath --from 'json(env://SECRETS)' --into github.actions
        env:
          SECRETS: ${{ toJSON(secrets) }}

      # downstream steps can now use the granted env vars
      - run: echo "token is masked in logs"
        env:
          GITHUB_TOKEN: ${{ env.GITHUB_TOKEN }}
```

the CLI handles:
- parse secrets JSON from `SECRETS` env var
- inject into process.env
- translate (e.g., github app JSON → ghs_* token)
- validate (block ghp_*, AKIA*, etc)
- output `::add-mask::` for each granted secret
- write to `$GITHUB_ENV` for downstream steps
- output `::notice::` for expiry times
- exit 2 if any blocked

### GITHUB_ENV multiline support

github actions requires heredoc syntax for multiline values in `$GITHUB_ENV`:

```typescript
// in getKeyrackFirewallOutput.ts, when into === 'github.actions'
const writeToGithubEnv = (key: string, value: string): void => {
  const githubEnv = process.env.GITHUB_ENV;
  if (!githubEnv) return;

  // detect multiline (PEM keys, JSON blobs)
  if (value.includes('\n')) {
    // heredoc syntax: KEY<<EOF\nVALUE\nEOF
    const delimiter = 'EOF_' + Date.now();
    appendFileSync(githubEnv, `${key}<<${delimiter}\n${value}\n${delimiter}\n`);
  } else {
    // simple syntax: KEY=VALUE
    appendFileSync(githubEnv, `${key}=${value}\n`);
  }
};
```

this covers:
- single-line values: `KEY=value`
- multiline PEM keys: heredoc with dynamic delimiter
- JSON blobs with newlines: heredoc syntax

### stdin implementation

when `--from 'json(stdin://*)'` is specified, read secrets from stdin:

```typescript
// in invokeKeyrackFirewall.ts
const readSecretsFromSource = async (source: KeyrackFirewallSource): Promise<Record<string, string>> => {
  let rawJson: string;

  if (source.type === 'env') {
    rawJson = process.env[source.envVar!] ?? '';
    if (!rawJson) {
      throw new ConstraintError('env var not set', {
        envVar: source.envVar,
        hint: `set ${source.envVar} or use --from 'json(stdin://*)'`,
      });
    }
  } else if (source.type === 'stdin') {
    // read all stdin
    const chunks: Buffer[] = [];
    for await (const chunk of process.stdin) {
      chunks.push(chunk);
    }
    rawJson = Buffer.concat(chunks).toString('utf8');
  } else {
    throw new UnexpectedCodePathError('unsupported source type', { source });
  }

  // parse JSON
  try {
    return JSON.parse(rawJson);
  } catch (error) {
    throw new ConstraintError('malformed secrets JSON', {
      source,
      hint: 'ensure the input is valid JSON object',
    });
  }
};
```

### getKeyrackFirewallOutput implementation

the output orchestrator formats grants for the target:

```typescript
// in getKeyrackFirewallOutput.ts
import type { KeyrackGrantAttempt, KeyrackKeyGrant } from '@src/domain.objects/keyrack';
import { ConstraintError } from 'helpful-errors';
import { appendFileSync } from 'fs';

export const getKeyrackFirewallOutput = (input: {
  attempts: KeyrackGrantAttempt[];
  grants: KeyrackKeyGrant[];
  into: 'github.actions' | 'json';
}): void => {
  // treestruct summary (always, for debug experience)
  console.log('');
  console.log('🔐 keyrack firewall');
  console.log(`   ├─ env: ${input.grants[0]?.env ?? 'unknown'}`);
  console.log(`   └─ grants`);
  for (const attempt of input.attempts) {
    const isLast = input.attempts.indexOf(attempt) === input.attempts.length - 1;
    const prefix = isLast ? '      └─' : '      ├─';
    if (attempt.status === 'granted') {
      const keyName = asKeyrackKeyName({ slug: attempt.grant.slug });
      const mech = attempt.grant.source.mech;
      const statusIcon = '🔑';
      console.log(`${prefix} ${keyName}`);
      console.log(`${prefix.replace(/[└├]/, ' ')}  ├─ mech: ${mech}`);
      if (attempt.grant.expiresAt) {
        console.log(`${prefix.replace(/[└├]/, ' ')}  ├─ expires: ${attempt.grant.expiresAt}`);
      }
      console.log(`${prefix.replace(/[└├]/, ' ')}  └─ status: granted ${statusIcon}`);
    } else if (attempt.status === 'blocked') {
      console.log(`${prefix} ${attempt.slug}`);
      console.log(`${prefix.replace(/[└├]/, ' ')}  ├─ status: blocked 🚫`);
      console.log(`${prefix.replace(/[└├]/, ' ')}  └─ reasons: ${attempt.reasons.join(', ')}`);
    } else if (attempt.status === 'absent') {
      console.log(`${prefix} ${attempt.slug}`);
      console.log(`${prefix.replace(/[└├]/, ' ')}  └─ status: absent 🫧`);
    }
  }
  console.log('');

  // format-specific output
  if (input.into === 'github.actions') {
    const githubEnv = process.env.GITHUB_ENV;
    if (!githubEnv) {
      throw new ConstraintError('GITHUB_ENV not set', {
        hint: '--into github.actions requires a GitHub Actions workflow context',
      });
    }

    for (const grant of input.grants) {
      const keyName = asKeyrackKeyName({ slug: grant.slug });
      const secret = grant.key.secret;

      // mask in logs
      console.log(`::add-mask::${secret}`);

      // expiry notice for ephemeral tokens
      if (grant.expiresAt) {
        console.log(`::notice::${keyName} expires at ${grant.expiresAt}`);
      }

      // write to GITHUB_ENV (heredoc for multiline)
      writeToGithubEnv(keyName, secret);
    }
  } else if (input.into === 'json') {
    // structured output for tests and programmatic use
    console.log(JSON.stringify(input.attempts, null, 2));
  }
};
```

## edge cases

| case | behavior |
|------|----------|
| secret not in manifest | skipped silently |
| all secrets granted | success, all exported |
| one secret blocked | fail fast, none exported (atomicity) |
| key absent (in manifest, not in secrets) | status: absent, no failure |
| empty keyrack.yml | success, no keys |
| empty secrets input | success, no keys |
| multiline secret value (PEM key) | handled via null-delimited parse |
| ephemeral token with expiry | expiry shown via ::notice:: |
| mech mismatch (manifest vs blob) | fail fast, ConstraintError with hint |
| blob has mech, manifest has none | use blob mech (inferred) |
| manifest has mech, blob is plain string | use manifest mech |
| --from env var not set | fail fast, ConstraintError |
| --from stdin empty | success, no keys (empty JSON object) |
