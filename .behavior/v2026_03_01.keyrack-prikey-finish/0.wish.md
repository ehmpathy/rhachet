  └───────────────┴────────────────┴────────────────────────────────────────┘

  Commands that don't need --prikey:
  - get - reads from daemon (no manifest decryption)
  - status - reads from daemon
  - relock - sends message to daemon

  So to answer your questions:

  1. list - yes, needs --prikey
  2. Also needs --prikey:
    - del
    - recipient get
    - recipient set
    - recipient del
