## subcomponent contracts

### roles spec parser

```
given('roles spec parser contract')
  then('exposes: parseRolesSpec(spec: string) => RolesSpec')
  then('RolesSpec has { mode: "replace" | "delta", ops: RoleOp[] }')
  then('RoleOp has { action: "add" | "remove", role: string }')
  then('throws BadRequestError for empty spec')
  then('throws BadRequestError for conflict (+foo,-foo)')
```

### role resolver

```
given('role resolver contract')
  then('exposes: resolveRoles(spec: RolesSpec, context: { agentDir: string }) => ResolvedRoles')
  then('ResolvedRoles has { roles: string[], errors: RoleError[] }')
  then('returns error for unknown role names with suggestion')
  then('throws BadRequestError if .agent/ not found')
  then('idempotent: +present role = no-op')
  then('idempotent: -absent role = no-op')
```

### config generator

```
given('config generator contract')
  then('exposes: genBrainConfig(roles: string[], brain: string) => ConfigPath')
  then('generates config with hooks for specified roles only')
  then('returns path to generated config file')
  then('respects brain-specific config format (e.g., claude-code settings.json)')
```

### brain spawner

```
given('brain spawner contract')
  then('exposes: spawnBrain(brain: string, config: ConfigPath, args: string[]) => void')
  then('spawns brain CLI with generated config')
  then('passes through all additional args to brain CLI')
  then('inherits stdin/stdout/stderr from parent process')
```

## composition boundaries

```
given('rhx enroll command implementation')
  then('composes: parser → resolver → generator → spawner')
  then('parser transforms --roles arg into structured spec')
  then('resolver computes final role list from spec and .agent/')
  then('generator produces brain-specific config file')
  then('spawner invokes brain CLI with config and passthrough args')
```

```
given('error flow')
  then('parser errors surface before resolver is invoked')
  then('resolver errors surface before generator is invoked')
  then('all errors prevent brain from boot')
  then('errors include actionable guidance')
```
