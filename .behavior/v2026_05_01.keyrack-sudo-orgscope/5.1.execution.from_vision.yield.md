## changes

### getAllSudoSlugsForKey.ts

added `repoOrg` parameter. when repoOrg provided, filter by `{org}.sudo.{key}` pattern. full slug still bypasses org filter. keys with `@all` org match regardless of repoOrg.

### unlockKeyrackKeys.ts

1. detect if key is full slug (contains `.` AND exists in hostManifest)
2. if not full slug, require repoManifest with org
3. if no repo manifest or no org declared, fail-fast with helpful hint
4. pass repoOrg to `getAllSudoSlugsForKey`

### unlockKeyrackKeys.integration.test.ts

updated 6 test contexts to include repo manifest with org for sudo unlock tests. this reflects the new behavior where sudo unlock requires repo org context (unless full slug).
