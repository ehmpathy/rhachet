## phase 3: 1password adapter

### prereqs
- phase 1 complete
- read `.behavior/v2026_03_24.fix-keyrack-vaults/3.3.1.blueprint.product.v1.i1.md` § codepath tree (1password)
- read `.behavior/v2026_03_24.fix-keyrack-vaults/3.3.1.blueprint.product.v1.i1.md` § contracts (vaultAdapter1Password)
- read `.behavior/v2026_03_24.fix-keyrack-vaults/3.1.1.research.external.product.access._.v1.i1.md`
- read `.behavior/v2026_03_24.fix-keyrack-vaults/3.1.3.research.internal.product.code.prod._.v1.i1.md`

### tasks
- [ ] implement `isOpCliInstalled()` — check `which op`
- [ ] implement `set()`:
  - [ ] check `isOpCliInstalled()`, throw constraint error if absent
  - [ ] prompt for exid (visible, not hidden)
  - [ ] validate roundtrip via `op read $exid`
  - [ ] return `{ exid }`
- [ ] implement `get()` — fetch via `op read $exid`, return secret or null
- [ ] return `mech=PERMANENT_VIA_REFERENCE` from set
- [ ] add adapter to vault map in `genContextKeyrackGrantUnlock.ts`

### acceptance
```
given(1password adapter implemented)
  given(op cli installed and authenticated)
    when(set --key K --vault 1password --env E)
      then(prompts for exid)
      then(validates roundtrip)
      then(returns exid)
      then(returns mech=PERMANENT_VIA_REFERENCE)
    when(get with valid exid)
      then(returns secret from 1password)
  given(op cli NOT installed)
    when(set --key K --vault 1password --env E)
      then(displays "op cli not found")
      then(displays install instructions)
      then(exits with code 2)
```

### verification
```sh
npm run test:types && npm run test:unit -- vaultAdapter1Password
```
