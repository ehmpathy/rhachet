├── [~] update: event determination
│   ├── [~] update: onBoot event selection via filter.what
│   │   ├── no filter → SessionStart (backwards compat)
│   │   ├── filter.what = valid boot event → that event

## supplier brief outline

```markdown
# brain hooks for PostCompact

## .what
run hooks only when context compacts, not on fresh session start.

## .why
PostCompact fires only on compaction. SessionStart fires on both.

## .how
```yaml
hooks:
  onBrain:
    onBoot:
      - command: npx rhachet run --init postcompact.verify.sh
        timeout: PT30S
        filter:
          what: PostCompact
```

## filter.what values
| value | when it fires |
|-------|---------------|
| (none) | SessionStart (backwards compat) |
| SessionStart | new session + compaction |
| PostCompact | compaction only |
| PreCompact | before compaction |
```
