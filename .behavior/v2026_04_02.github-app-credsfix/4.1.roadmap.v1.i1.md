# roadmap: github app credentials fix

## overview

this roadmap executes the blueprint at `.behavior/v2026_04_02.github-app-credsfix/3.3.1.blueprint.product.v1.i1.md`

**goal**: fix stdin truncation so multiline json (with embedded newlines like RSA keys) round-trips correctly through keyrack set/get.

**root cause**: `readline.once('line', ...)` reads only the first line of piped stdin.

**fix**: replace readline with async iterator pattern to read all stdin content.

- [ ] preserve final newline trim: `content.endsWith('\n') ? content.slice(0, -1) : content`
- [ ] leave TTY branch untouched

**code after** (piped stdin branch):
```ts
if (!process.stdin.isTTY) {
  const chunks: string[] = [];
  process.stdin.setEncoding('utf8');
  for await (const chunk of process.stdin) {
    chunks.push(chunk as string);
  }
  const content = chunks.join('');
  return content.endsWith('\n') ? content.slice(0, -1) : content;
}
```
