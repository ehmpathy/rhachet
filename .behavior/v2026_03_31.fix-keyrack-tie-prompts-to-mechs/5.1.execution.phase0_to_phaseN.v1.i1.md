## key changes summary

| before | after |
|--------|-------|
| vault.set prompted for secrets | mech.acquireForSet prompts, vault stores |
| vault.get returned source | vault.get calls mech.deliverForGet, returns translated |
| vaults knew about mech prompts | vaults are pure storage backends |
| mechs only had translate | mechs have acquireForSet + deliverForGet |
| aws.iam.sso vault name | aws.config vault name |
| inferMechFromVault (one mech per vault) | inferKeyrackMechForSet (prompt if multiple) |
