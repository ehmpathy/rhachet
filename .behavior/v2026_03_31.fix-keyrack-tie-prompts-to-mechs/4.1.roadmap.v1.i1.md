## phase 3: domain operations

**goal:** update set flow with vault/mech inference and encapsulation

**read before:**
- `.behavior/v2026_03_31.fix-keyrack-tie-prompts-to-mechs/3.3.1.blueprint.product.v1.i1.md` (codepath tree sections)
- `.agent/repo=.this/role=keyrack/briefs/define.vault-mech-adapters.md`

### checklist

- [ ] **3.1** update `inferKeyrackVaultFromKey`
  - rename `aws.iam.sso` → `aws.config` in inference
  - location: `src/domain.operations/keyrack/inferKeyrackVaultFromKey.ts`

- [ ] **3.2** create `inferKeyrackMechForSet`
  - input: `{ vault: KeyrackHostVaultAdapter }`
  - if single mech supported → auto-select
  - if multiple mechs → prompt via stdin
  - return selected mech
  - location: `src/domain.operations/keyrack/inferKeyrackMechForSet.ts`

- [ ] **3.3** update vault adapter `set` to encapsulate mech calls
  - vault.set internally:
    1. infer mech if not supplied (via inferKeyrackMechForSet)
    2. check mech compat (fail-fast if not in mechs.supported)
    3. call mech.acquireForSet({ keySlug })
    4. store source credential
  - secret never leaves vault scope

- [ ] **3.4** update vault adapter `get` to encapsulate mech delivery
  - vault.get internally:
    1. retrieve source from storage
    2. call mech.deliverForGet({ source })
    3. return translated secret

- [ ] **3.5** simplify `setKeyrackKey`
  - vault inference (if --vault absent)
  - vault lookup
  - call vault.set({ slug, mech?, exid? })
  - roundtrip validation
  - location: `src/domain.operations/keyrack/setKeyrackKey.ts`

- [ ] **3.6** delete obsolete files
  - delete `inferMechFromVault.ts` (incompatible: assumed one mech per vault)

### verification

```
given(inferKeyrackMechForSet implemented)
  when(vault supports single mech)
    then(auto-selects, no prompt)
  when(vault supports multiple mechs)
    then(prompts via stdin with options)

given(vault.set encapsulates mech calls)
  when(set called without --mech)
    then(infers mech internally)
  when(set called with explicit --mech)
    then(validates mech compat, fails fast if incompatible)
  when(mech.acquireForSet completes)
    then(source credential stored, never exposed to caller)

given(vault.get encapsulates mech delivery)
  when(get called for EPHEMERAL_VIA_GITHUB_APP)
    then(retrieves json blob, transforms to ghs_ token)
  when(get called for PERMANENT_VIA_REPLICA)
    then(retrieves source, returns unchanged (identity))
```
