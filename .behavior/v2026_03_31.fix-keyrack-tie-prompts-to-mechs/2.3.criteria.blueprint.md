## subcomponent contracts

### mech adapter contract

```
given('mech adapter contract')
  then('each mech has adapter: MechAdapter')
  then('adapter.name identifies the mech')
  then('adapter.promptForSet runs guided setup via stdin, returns source credential')
  then('adapter.transform takes source credential, returns usable secret')
  then('permanent mechs: transform is identity')
  then('ephemeral mechs: transform produces short-lived token')
```

### vault adapter contract

```
given('vault adapter contract')
  then('each vault has adapter: VaultAdapter')
  then('adapter.name identifies the vault')
  then('adapter.supportedMechs lists compatible mechs')
  then('adapter.checkMechCompat validates mech, fails fast with alternatives')
  then('adapter.set stores source credential')
  then('adapter.get retrieves source credential')
```

### mech inference contract

```
given('mech inference contract')
  then('domain.operation: inferMech')
  then('input: vault adapter, key name')
  then('output: selected mech name')
  then('when vault supports one mech: auto-select')
  then('when vault supports many mechs: prompt via stdin')
```

### vault inference contract

```
given('vault inference contract')
  then('domain.operation: inferVault')
  then('input: key name')
  then('output: vault name or null')
  then('AWS_PROFILE, AWS_* patterns: returns aws.config')
  then('other patterns: returns null (require explicit --vault)')
```

## composition boundaries

### keyrack set flow

```
given('keyrack set implementation')
  then('composes: vault inference → vault lookup → mech inference → mech adapter → vault storage')
  then('vault inference: infers --vault if not supplied')
  then('vault lookup: retrieves vault adapter by name')
  then('vault adapter: checks mech compatibility')
  then('mech inference: prompts if --mech not supplied and multiple mechs valid')
  then('mech adapter: runs promptForSet, produces source credential')
  then('vault adapter: stores source credential via set method')
```

### keyrack unlock flow

```
given('keyrack unlock implementation')
  then('composes: vault retrieval → mech transformation → daemon storage')
  then('vault adapter: retrieves source credential via get method')
  then('mech adapter: transforms source credential to usable secret')
  then('daemon: stores translated secret')
```

## integration boundaries

```
given('mech adapter boundary')
  then('mechs are independent of vault storage format')
  then('same mech adapter works with any compatible vault')
    sothat('EPHEMERAL_VIA_GITHUB_APP works with os.secure and 1password')

given('vault adapter boundary')
  then('vaults are independent of mech prompt logic')
  then('vault stores whatever mech produces')
    sothat('vaults remain pure storage backends')

given('stdin prompt boundary')
  then('all user interaction via stdin prompts')
  then('mech adapters own prompt content')
  then('mech inference owns mech selection prompt')
    sothat('prompts are testable and consistent')
```
