## pattern 6: opencode plugin filename regex [EXTEND]

```typescript
export const parsePluginFileName = (
  filename: string,
): { author: string; event: string } | null => {
  const match = filename.match(
    /^rhachet-(.+)-(onBoot|onTool|onStop)-[a-zA-Z0-9_-]+\.ts$/,
  );
  if (!match) return null;
  return {
    author: match[1]!,
    event: match[2]!,
  };
};
```

**relation to wish:**
- matches only onBoot|onTool|onStop
- must add onTalk to regex pattern

**action:** extend regex to include onTalk
