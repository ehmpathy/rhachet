# blueprint: add onTalk hook to Role.build

## summary

add `onTalk` event to rhachet's hook system:
- extend `BrainHookEvent` type with `'onTalk'`
- extend `RoleHooksOnBrain` interface with `onTalk` property
- extend `extractDeclaredHooks` to process onTalk hooks
- extend claude code adapter to map `onTalk` → `UserPromptSubmit`
- extend opencode adapter to map `onTalk` → `chat.message`
- extend tests for all touch points
