## foreman notes

this is a type-level change. the runtime behavior of tool calls depends on brain suppliers (anthropic, openai) that implement the contract.

rhachet core provides:
1. type definitions (BrainPlugToolDefinition, BrainPlugToolInvocation)
2. conditional type narrower (AsBrainOutputCallFor, AsBrainOutputOutputFor)
3. extended BrainAtom.ask signature with TPlug generic

brain suppliers consume this contract. runtime verification occurs in supplier packages.
