# blackbox criteria: BrainAtom tool support

> what experience must be delivered to fulfill the wish?

---

# usecase.1 = agentic brain orchestrator

the brain orchestrates tool calls to fulfill a task.

## episode: tool call loop

```
given('a brain with tools plugged')
  when('the brain needs external data to answer')
    then('the response includes call.tools with invocations')
      sothat('the caller can execute tools and continue')
    then('each invocation has exid, slug, and input')
      sothat('the caller can correlate results')
    then('output may be null')
      sothat('the caller knows the brain deferred to tools')

  when('the caller continues with tool results as input')
    then('the brain receives the tool output in the next exchange')
      sothat('it can incorporate the data into its answer')
    then('the brain may request more tool calls')
      sothat('multi-step workflows are possible')
    then('the brain may return final output')
      sothat('the episode completes')

  when('the brain can answer directly without tools')
    then('output is populated with the answer')
    then('call is null')
      sothat('the caller knows no tools were needed')
```

## episode: parallel tool calls

```
given('a brain with multiple tools plugged')
  when('the brain needs data from multiple sources')
    then('call.tools contains multiple invocations')
      sothat('the caller can execute them in parallel')
    then('each invocation is independent')
      sothat('execution order does not matter')
```

# usecase.2 = structured extraction with fallback

the brain extracts structured data, with tool access for absent info.

## episode: direct extraction

```
given('a brain with tools plugged and a schema.output defined')
  when('the brain can satisfy the schema from context alone')
    then('output matches the schema')
    then('call is null')
      sothat('no tools were needed')
```

## episode: tool-assisted extraction

```
given('a brain with tools plugged and a schema.output defined')
  when('the brain needs tool data to satisfy the schema')
    then('call.tools contains the invocation')
    then('output is null')
      sothat('the caller knows the schema is not yet satisfied')

  when('the caller continues with tool result')
    then('output matches the schema')
      sothat('the brain incorporated the tool data')
```

# usecase.3 = backwards compatibility

extant callers without tools continue to work unchanged.

## exchange: no tools plugged

```
given('a brain without tools plugged')
  when('atom.ask() is called')
    then('output is always TOutput (never null)')
      sothat('extant callers need no null checks')
    then('call is null')
      sothat('no tool call machinery is exposed')
```

# exchange contracts

## input: plug.tools

```
given('tools are plugged into atom.ask()')
  when('each tool has slug, description, schema.input')
    then('the brain can invoke any plugged tool')
    then('the brain cannot invoke tools not plugged')
      sothat('the caller controls available capabilities')
```

## output: call.tools

```
given('tools are plugged')
  when('the brain requests tool invocations')
    then('call.tools is an array of ToolInvocation')
    then('each ToolInvocation has exid, slug, input')
    then('input matches the tool's schema.input')
      sothat('the caller can validate and execute')

  when('the brain does not request tools')
    then('call is null')
```

## output: output with tools

```
given('tools are plugged')
  when('the brain returns only tool calls')
    then('output is null')
  when('the brain returns both output and tool calls')
    then('output is TOutput')
    then('call.tools is populated')
      sothat('the caller can process both')
  when('the brain answers directly')
    then('output is TOutput')
    then('call is null')
```

# edge cases

## unknown tool requested

```
given('tools are plugged')
  when('the brain attempts to call a tool not in plug.tools')
    then('the invocation is not included in call.tools')
      sothat('the brain cannot escape the plugged toolset')
```

## invalid tool input

```
given('a tool with schema.input defined')
  when('the brain provides input that violates the schema')
    then('the invocation is included with the raw input')
      sothat('the caller can validate and handle the error')
```

## empty tools array

```
given('plug.tools is an empty array')
  when('atom.ask() is called')
    then('the brain has no tools available')
    then('call is null')
    then('output is TOutput | null')
      sothat('the type system reflects tools-aware mode')
```
