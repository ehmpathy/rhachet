### step 3: analyze the gap

the two uncovered edge cases are runtime behaviors:
- **unknown tool**: requires actual LLM to attempt the call
- **invalid tool input**: requires actual LLM to provide malformed input

rhachet core does not execute LLM calls — that's what brain suppliers do. the playtest can only verify:
- type-level behaviors (conditional types, narrower)
- file shapes (domain objects)
- backwards compatibility (acceptance tests)

the "foreman notes" section documents this scope boundary.
