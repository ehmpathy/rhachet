## deeper investigation: BrainReplPlugs vs BrainPlug

i found `BrainReplPlugs.ts` which has a `toolboxes?: never` field. could this duplicate our `BrainPlug.tools`?

**reviewed**: BrainReplPlugs.ts:9-17

```ts
toolboxes?: never; // todo: allow configuration
// toolboxes?: BrainActorToolbox[];
```

**conclusion**: these are different concepts:
- `BrainReplPlugs.toolboxes` = tool *providers* (where tools come from, e.g., MCP servers)
- `BrainPlug.tools` = tool *definitions* (what tools are available, e.g., function schemas)

`BrainReplPlugs` is repl-specific (toolboxes, memory, access). `BrainPlug` is shared across atoms and repls. no duplication.
