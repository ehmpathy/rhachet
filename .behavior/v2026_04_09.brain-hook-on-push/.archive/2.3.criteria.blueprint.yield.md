## subcomponent contracts

### onPush hook contract

```
given('RoleHooksOnBrain.onPush contract')
  then('accepts array of OnPushHook')
  then('each OnPushHook has filter: { topic: string, roles: string }')
  then('each OnPushHook has command: string')
  then('each OnPushHook optionally has timeout: string (ISO 8601 duration)')
```

### message contract

```
given('BrainHookPushMessage contract')
  then('has id: string')
  then('has for: { topic: string, roles: string }')
  then('has timestamp: string (ISO 8601)')
  then('has payload: unknown')
  then('is stored as JSON file at .agent/.comms/$topic/$id.json')
```

### role filter contract

```
given('BrainHookPushFilter contract')
  then('parses +role as inclusion')
  then('parses -role as exclusion')
  then('parses comma-separated patterns')
  then('returns boolean match result')
```

### poll adapter contract

```
given('PollAdapter contract')
  then('watches .agent/.comms/ for new files')
  then('parses each file as BrainHookPushMessage')
  then('matches message.for against registered hook filters')
  then('invokes hook command for matched messages')
  then('removes or moves processed messages')
  then('moves malformed messages to .comms/.errors/')
  then('moves failed messages to .comms/.errors/')
```

### poll task lifecycle contract

```
given('PollTaskLifecycle contract')
  then('ensures poll task is active when clone is active')
  then('restarts poll task if it exits unexpectedly')
  then('uses heartbeat to detect stale poll task')
  then('integrates with onStop hook to force restart via exit(2)')
```

## composition boundaries

```
given('onPush feature implementation')
  then('role hooks declare onPush handlers')
  then('rhx init --hooks registers poll task in brain session')
  then('poll adapter watches .comms/ and invokes hooks')
  then('poll task lifecycle ensures continuous operation')

given('message delivery flow')
  then('external system writes to .agent/.comms/$topic/$id.json')
  then('poll adapter detects new file')
  then('poll adapter parses message')
  then('poll adapter matches filter against clone roles')
  then('poll adapter invokes hook command if matched')
  then('poll adapter outputs to stdout to wake clone')
  then('clone processes notification')
```

## integration boundaries

```
given('brain adapter integration')
  then('onPush hooks are translated to brain-specific mechanism')
  then('for claude code: background task + onStop hook')
  then('other brains may use different mechanisms')

given('external system integration')
  then('external systems write JSON files to .agent/.comms/')
  then('no direct integration with brain required')
  then('filesystem is the interface')
```
