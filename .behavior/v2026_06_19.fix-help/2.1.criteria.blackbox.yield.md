# blackbox criteria: rhx help & list

## usecase.1 = get help

```
given(rhachet is installed)
  when(user runs `rhx --help`)
    then(shows commands treestruct)
      sothat(user discovers available commands)
    then(shows `ref: $readme` at bottom)
      sothat(user can learn more)

  when(user runs `rhx -h`)
    then(same output as `rhx --help`)
      sothat(unix standard short flag works)

  when(user runs `rhx help`)
    then(same output as `rhx --help`)
      sothat(subcommand form works)

  when(user runs `rhx --help garbage`)
    then(`--help` takes precedence, shows help)
      sothat(flags override positional args)
```

## usecase.2 = list skills

```
given(roles linked with N skills)
  when(user runs `rhx list`)
    then(shows "N skills found" as first branch)
    then(shows treestruct grouped by repo > role > skill)
      sothat(user sees available skills organized by source)

given(no roles linked)
  when(user runs `rhx list`)
    then(shows "0 skills found")
    then(shows hint to run `rhachet roles link`)
      sothat(user knows how to get skills)
```

## usecase.3 = search skills by pattern

```
given(4 skills contain "radio")
  when(user runs `rhx list radio`)
    then(header renders as `rhx list '*radio*'`)
      sothat(user sees what "contains" means)
    then(shows "4 skills found")
    then(shows only skills with "radio" in name)

given(4 skills match glob `git.commit.*`)
  when(user runs `rhx list 'git.commit.*'`)
    then(header renders as `rhx list git.commit.*`)
    then(shows "4 skills found")
    then(shows only skills that match glob)

given(no skills match pattern)
  when(user runs `rhx list nonexistent`)
    then(shows "0 skills found")
```

## usecase.4 = filter skills

```
given(skills in multiple repos)
  when(user runs `rhx list --repo ehmpathy`)
    then(shows only skills from ehmpathy repo)
      sothat(user can narrow scope)

given(skills in multiple roles)
  when(user runs `rhx list --role mechanic`)
    then(shows only skills from mechanic role)
      sothat(user can narrow scope)

given(skills matchable by pattern and repo)
  when(user runs `rhx list radio --repo bhuild`)
    then(shows only radio skills in bhuild)
      sothat(filters stack)
```

## usecase.5 = truncation

```
given(30 skills in one role)
  when(user runs `rhx list`)
    then(shows first 10 skills)
    then(shows "... (20 more, use --all to see all)")
      sothat(output is scannable and user knows how to see all)

  when(user runs `rhx list --all`)
    then(shows all 30 skills)
    then(no truncation message)
      sothat(user can see full list when needed)
```

## usecase.6 = list-specific help

```
given(rhachet is installed)
  when(user runs `rhx list --help`)
    then(shows list-specific options treestruct)
    then(shows [pattern], --repo, --role, --all options)
      sothat(user learns list command usage)
```
