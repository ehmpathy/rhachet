## pattern.1 = bin/rhx special-case dispatch

**[EXTEND]**

the `bin/rhx` shell dispatcher proxies to `rhachet run --skill`, but first short-circuits certain commands to avoid skill dispatch overhead.

### citation.1.1 — short-circuit keyrack

```sh
# short-circuit keyrack commands (via jit path, not bun binary)
if [ "$1" = "keyrack" ]; then
  shift
  exec "$SCRIPT_DIR/run.jit" keyrack "$@"
fi
```

### citation.1.2 — short-circuit upgrade

```sh
# short-circuit upgrade command (top-level rhachet command, not a skill)

fi
```

source: `bin/rhx:33-37`

### citation.1.3 — short-circuit enroll

```sh
# short-circuit enroll command (top-level rhachet command, not a skill)
if [ "$1" = "enroll" ]; then
  shift
  exec "$SCRIPT_DIR/run" enroll "$@"
fi
```

source: `bin/rhx:39-43`

### citation.1.4 — default proxy to skill

```sh
# default: proxy to run --skill
exec "$SCRIPT_DIR/run" run --skill "$@"
```

source: `bin/rhx:45-46`

### how we will extend

add help and list intercepts before the default proxy:

```sh
# short-circuit help command
if [ "$1" = "help" ] || [ "$1" = "--help" ] || [ "$1" = "-h" ]; then
  exec "$SCRIPT_DIR/run" run --help
fi

# short-circuit list command
if [ "$1" = "list" ]; then
  shift
  exec "$SCRIPT_DIR/run" run list "$@"
fi
```
