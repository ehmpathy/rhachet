│  │  ### 3. BLOCKER — Naming collision: two `list` commands, different semantics
│  │  
│  │  | command | handler | what it lists | requires config |
│  │  |---------|---------|---------------|-----------------|
│  │  | `rhachet list` | `invokeList` | roles/repos from npm registry | yes |
