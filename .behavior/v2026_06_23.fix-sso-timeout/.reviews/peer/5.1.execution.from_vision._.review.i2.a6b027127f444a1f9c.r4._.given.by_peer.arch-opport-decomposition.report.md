# blocker.4: Mixed grains in `validateSsoSession` - combines I/O and computation

**rule**: .agent/repo=ehmpathy/role=architect/briefs/practices/rule.prefer.decomposable-architecture.md

**locations**:
- src/domain.operations/keyrack/adapters/vaults/aws.config/vaultAdapterAwsConfig.ts:65-115

The `validateSsoSession` function in `vaultAdapterAwsConfig.ts` mixes communicator (I/O: fromSSO, export-credentials, sts call) and transformer (parsing ARN to extract username, error message analysis) responsibilities. This violates decomposable architecture. Split into separate leaf operations: e.g., `validateSsoTokenWithAwsSdk` (communicator), `refreshStsCredentials` (communicator), `getCallerIdentity` (communicator), and `extractUsernameFromArn` (transformer). The orchestrator should then compose these named operations.

**snippet**:
```ts
const validateSsoSession = async (
  profileName: string,
): Promise<{ valid: true; username: string } | { valid: false }> => {
  try {
    await validateSsoTokenWithAwsSdk(profileName);
  } catch (error) {
    if (
      error instanceof Error &&
      (error.name === 'CredentialsProviderError' ||
        error.message.includes('expired') ||
        error.message.includes('invalid'))
    ) {
      return { valid: false };
    }
    throw new UnexpectedCodePathError(
      'validateSsoTokenWithAwsSdk failed with unexpected error',
      { profileName, error },
    );
  }

  try {
    await execAsync(
      `aws configure export-credentials --profile "${profileName}" --format env-no-export`,
    );
  } catch (error) {
    const message = error instanceof Error ? error.message.toLowerCase() : '';
    const isExpiredError =
      message.includes('expired') ||
      message.includes('invalid') ||
      message.includes('unauthorized') ||
      message.includes('does not exist') ||
      message.includes('refreshed credentials are still expired');

    if (isExpiredError) return { valid: false };
    throw new UnexpectedCodePathError(
      'aws configure export-credentials failed with unexpected error',
      { profileName, error },
    );
  }

  try {
    const { stdout } = await execAsync(
      `aws sts get-caller-identity --profile "${profileName}"`,
    );
    const identity = JSON.parse(stdout) as { Arn: string };
    const arnParts = identity.Arn.split('/');
    const username = arnParts[arnParts.length - 1] ?? 'unknown';
    return { valid: true, username };
  } catch (error) {
    if (error instanceof UnexpectedCodePathError) throw error;
    if (error instanceof ConstraintError) throw error;
    if (error instanceof Error && 'code' in error) return { valid: false };
    throw error;
  }
};
```
