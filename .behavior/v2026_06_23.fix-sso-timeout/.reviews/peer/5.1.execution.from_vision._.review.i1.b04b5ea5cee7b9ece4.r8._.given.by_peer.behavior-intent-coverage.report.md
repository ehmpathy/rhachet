# blocker.1: Unhandled SSO timeout in setupAwsSsoProfile function

**rule**: .agent/repo=bhuild/role=behaver/briefs/practices/behavior.execution/architect/rule.require.behavior-intent-coverage.md

**locations**:
- src/domain.operations/keyrack/adapters/mechanisms/aws.sso/setupAwsSsoProfile.ts:417-420

The setupAwsSsoProfile function executes 'aws sso login' via execSync and catches any error with UnexpectedCodePathError without distinguishing timeout from other failures. According to the wish and vision, a timeout should produce a ConstraintError with the treestruct error message to instruct the robot to halt and ask the human. This gap means that when setupAwsSsoProfile is called (e.g., during initial profile setup or when using the --exid case), a timeout will not produce the loud, helpful error but instead a generic UnexpectedCodePathError, which leads to wasted tokens and confusion.

**snippet**:
```ts
try {
    execSync(`aws sso login --profile "${input.profileName}"`, {
      stdio: 'pipe',
    });
  } catch {
    throw new UnexpectedCodePathError('aws sso login failed', {
      profileName: input.profileName,
    });
  }
```

# blocker.2: Non-timeout failures incorrectly classified as ConstraintError in triggerSsoLogin

**rule**: .agent/repo=bhuild/role=behaver/briefs/practices/behavior.execution/architect/rule.require.behavior-intent-coverage.md

**locations**:
- src/domain.operations/keyrack/adapters/vaults/aws.config/vaultAdapterAwsConfig.ts:580-590

In vaultAdapterAwsConfig.ts, the triggerSsoLogin function throws ConstraintError for all non-timeout failures (e.g., network errors, aws cli not installed). The vision explicitly requires that only timeout results in ConstraintError; other failures should be MalfunctionError. This misclassification leads to incorrect robot behavior: the robot will halt and ask the human for network issues that the robot should handle.

**snippet**:
```ts
} else {
          reject(
            new ConstraintError('aws sso login failed', {
              profileName,
              exitCode: code,
              output: outputBuffer,
              hint: 'complete browser auth to continue',
            }),
          );
```
