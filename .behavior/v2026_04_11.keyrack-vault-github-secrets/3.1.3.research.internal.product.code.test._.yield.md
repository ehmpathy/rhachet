## pattern.4 = snapshot testing for stdout

**[REUSE]** — same strip + snapshot approach

### citation.4

file: `blackbox/cli/keyrack.vault.osSecure.githubApp.acceptance.test.ts` lines 139-153

```typescript
then('stdout matches snapshot', () => {
  const stripped = result.stdout
    .replace(/\x1B\[[0-9;]*[A-Za-z]/g, '') // ANSI escape sequences
    .replace(/\x1B\]/g, '') // OSC sequences
    .replace(/\r/g, '') // carriage returns from PTY
    .replace(/·/g, '') // middle dots from PTY
    .replace(/\s+$/gm, ''); // trim end-of-line whitespace
  const treeStart = stripped.indexOf('\u{1F510}');
  const clean = stripped.slice(treeStart >= 0 ? treeStart : 0).trim();
  expect(clean).toMatchSnapshot();
});
```

### reuse pattern

github.secrets tests will use identical snapshot stripping
